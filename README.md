# Starus Bot
- A custom build slack bot built using JS available 24/7 and deployed on Slack

## Commands Available 
- /mood - asks you about your mood
- /tellmeajoke - tells you a joke
- /givemeaquote - Gives you random motivational quote

## Tech Stack
- JS

## Set up and install
- look up Starus app on Slack app or just download all the files and host it yourself on any suitable hosting platform
